📊 Projeto simples feito com HTML, CSS e JavaScript, desenvolvido para fins estudantis, com foco em praticar conceitos do front-end.

🚀 Funcionalidades
➕ Aumentar número
➖ Diminuir número
🔄 Resetar contador
🌙 Alternar modo escuro
🧠 Como funciona
Uma variável (contador) guarda o valor
Botões chamam funções (aumentar, diminuir, resetar)
O valor é atualizado na tela usando o DOM (innerText)
O tema é alterado com classList.toggle()
🎨 Estilo
Layout centralizado com flexbox
Cores modernas (azul, branco e cinza escuro)
Efeitos de hover e clique
Modo escuro com classe .dark
📱 Responsivo
Funciona em celular e desktop
Usa media queries para ajustar tamanhos
📁 Estrutura
index.html
style.css
script.js

Projeto básico voltado para aprendizado, ideal para praticar:

lógica em JavaScript
